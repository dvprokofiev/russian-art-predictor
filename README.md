# Определитель принадлежности картинки к стилю русского искусства

Модель помогает определить, к какому стилю живописи/искусства относится изображенное на фото. Она умеет различать (пока что):

- иконы
- парсуны
- барокко
- русский ампир
- романтизм
- академизм
- передвижников
- модерн
- суприматизм
- конструктизм
- авангард (как все, что не относится к суприматизму и конструктивизму)

## Сама модель
Модель можно скачать с [Hugging Face](https://huggingface.co/dvprokofiev/russian-art-predictor)

## Датасет
Датасет, на котором обучается модель, можно скачать [здесь](https://huggingface.co/datasets/dvprokofiev/russian-art)
